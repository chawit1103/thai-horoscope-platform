# BETA_USER_FAQ.md - Beta User FAQ

## Goal

Give beta users clear Thai answers about scope, privacy, limitations, subscriptions, notifications, and feedback.

## FAQ

### beta นี้คืออะไร

```text
beta คือช่วงทดลองใช้งานก่อนเปิดบริการกว้างขึ้น เราใช้ช่วงนี้เพื่อตรวจสอบความเข้าใจง่ายของ onboarding การตั้งค่าข้อมูลเกิด การดู horoscope และการตั้งค่าการแจ้งเตือน
```

### เนื้อหา horoscope ใช้เพื่ออะไร

```text
เนื้อหาใช้เพื่อความบันเทิง การสะท้อนตนเอง และการวางแผนชีวิตประจำวันอย่างมีสติ ไม่ใช่คำทำนายที่รับประกันผลลัพธ์ และไม่ใช่คำแนะนำทางการแพทย์ การเงิน หรือกฎหมาย
```

### ถ้าไม่ทราบเวลาเกิด ใช้ได้ไหม

```text
ใช้ได้ คุณสามารถเลือก “ไม่ทราบเวลาเกิด” ได้ ระบบจะทำให้ข้อความบางส่วนเป็นแนวโน้มกว้าง ๆ มากขึ้น และหลีกเลี่ยงการตีความที่ต้องใช้เวลาเกิดอย่างละเอียด
```

### beta ทำให้ได้สิทธิ์ premium หรือไม่

```text
ไม่เสมอไป สิทธิ์ beta ใช้สำหรับเข้าร่วมการทดลองบางส่วน แต่สิทธิ์รายเดือน รายปี หรือ premium ยังขึ้นอยู่กับ subscription และ entitlement ของบัญชี
```

### จะได้รับ Email หรือ LINE จริงไหม

```text
ช่วง beta อาจใช้ sandbox, dry-run, หรือ staging/test mode ตามที่ทีมอนุมัติ ระบบทดสอบต้องไม่ส่ง Email หรือ LINE จริงโดยไม่ได้รับอนุมัติจากผู้ดูแล
```

### ข้อมูลส่วนตัวถูกใช้ยังไง

```text
ข้อมูลที่จำเป็นจะถูกใช้เพื่อการตั้งค่าบัญชี การสร้างประสบการณ์ horoscope และการแจ้งเตือนที่คุณเลือกไว้ คุณสามารถขอ export ข้อมูล ลบข้อมูลเกิด ขอปิดบัญชี หรือ unsubscribe จากการแจ้งเตือนได้
```

### ควรส่ง feedback แบบไหน

```text
ส่ง feedback เกี่ยวกับความเข้าใจง่าย ความน่าใช้ ความชัดเจนของข้อความ ปัญหาการตั้งค่า และสิ่งที่ควรปรับปรุง

กรุณาไม่ส่งข้อมูลลับหรือข้อมูลส่วนตัวแบบเต็ม เช่น email แบบเต็ม LINE user ID รหัส invite แบบเต็ม วันเวลาเกิดแบบละเอียด ข้อมูลการชำระเงิน token หรือ secret
```

### ถ้าเจอข้อความที่ดูน่ากังวลควรทำอย่างไร

```text
โปรดส่ง feedback พร้อมอธิบายว่าข้อความส่วนไหนทำให้รู้สึกไม่สบายใจ ทีมจะตรวจสอบเพื่อปรับให้เหมาะสมขึ้น เนื้อหาไม่ควรกดดัน ทำให้กลัว หรืออ้างผลลัพธ์ที่แน่นอน
```

### ถ้าต้องการหยุดใช้งาน beta

```text
คุณสามารถปิดการแจ้งเตือน unsubscribe ลบข้อมูลเกิด หรือขอปิดบัญชีได้จากหน้าการตั้งค่าหรือช่องทาง support ที่กำหนด
```

## User Safety Reminder

```text
โปรดใช้เนื้อหาเป็นแนวทางทบทวนตัวเองเท่านั้น หากเป็นเรื่องสุขภาพ การเงิน กฎหมาย หรือความปลอดภัย ควรปรึกษาผู้เชี่ยวชาญที่เหมาะสม
```
